# Раздел 6: Мультиагентные системы

## Обзор

В этом разделе мы переходим от одиночных агентов к **мультиагентным системам** с использованием библиотеки **OpenAI Agents SDK**. Мы создаём **Deep Research Agent** — агента с агентским циклом (agentic loop) для глубокого исследования тем, изучаем мониторинг через RunHooks и Streaming Events, а затем рассматриваем три паттерна координации агентов.

## Ключевые темы

- **OpenAI Agents SDK** с Yandex Cloud (AsyncOpenAI, OpenAIResponsesModel)
- Агентский цикл (ReAct): планирование → действие → анализ → повтор
- Создание инструментов через декоратор `@function_tool`
- `Runner.run()` для запуска агентского цикла
- **RunHooks** — мониторинг действий агента через коллбэки
- **Streaming Events** — потоковый вывод токен за токеном через `Runner.run_streamed()`

### Паттерны координации мультиагентных систем

| Паттерн | Контроль | Контекст | Когда использовать |
|---------|----------|----------|-------------------|
| **Sequential Chain** | Последний агент | Полная передача | Простые конвейеры |
| **Hub-and-Spoke** (Координатор) | LLM-координатор | Полная передача | Динамические workflow |
| **Agents-as-Tools** | LLM-координатор | Новый контекст | Сложная оркестрация |
