# Раздел 3: Function Calling (Tool Calling)

## Обзор

В этом разделе мы изучаем механизм **Function Calling** (вызов функций), который позволяет LLM взаимодействовать с внешними инструментами. Мы рассматриваем два способа определения инструментов — через JSON Schema и через **Pydantic**, строим универсальный класс `Agent` и создаём исследовательского агента для работы с arXiv API.

## Ключевые темы

- Механизм Function Calling: как LLM вызывает внешние функции
- Определение инструментов через JSON Schema
- Определение инструментов через Pydantic-модели (автоматическое генерирование схемы)
- Полный цикл взаимодействия LLM ↔ инструменты
- Универсальный класс `Agent` с поддержкой цикла вызова функций
- Работа с arXiv API: поиск статей, получение деталей, анализ
