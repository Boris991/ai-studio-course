# Раздел 4: RAG и поиск

## Обзор

В этом разделе мы изучаем два встроенных инструмента Responses API для расширения знаний LLM: **Web Search** (поиск в интернете) и **File Search** (поиск по загруженным документам). Мы создаём персональную базу знаний на основе научных статей, сравниваем подходы RAG и Tool Calling, а также интегрируем агента с Telegram-ботом.

## Ключевые темы

- **Web Search**: инструмент поиска в интернете через Responses API
- Суммаризация новостей, конкурентный анализ
- **File Search**: загрузка файлов, создание Vector Store, индексация
- Стратегии чанкования документов
- Универсальный класс [`Agent.py`](Agent.py) с поддержкой всех типов инструментов (Web Search, File Search, MCP, Pydantic)
- Интеграция с Telegram-ботом для управления базой знаний
